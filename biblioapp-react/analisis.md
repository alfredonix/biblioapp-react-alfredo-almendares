Componente
Se utiliza desde un principio, es decir, sin este no se tomaría ningún dato de libros, nos permite encapsular datos, anidar con otros componentes.

Es el adecuado ya que con su forma de guardar los datos y poder anidar sin complejidad con otros componentes, es decir, no rompiendo su propia lógica con esto nos facilita la tarea de crear UIs en todo sentido, ya sea complejas, extensas, también como simples.

JSX

Este se utiliza para guardar los componentes obviamente con su lógica propia y su función especifica dejándonos añadir directamente variables de JavaScript.

Es el adecuado ya que sin este el trabajo sería mucho más complejo y con mucho más texto.


Estado (useState)

Este se utiliza para diversas cosas, ya sea para importar bibliotecas como para ayudarnos a mantener datos guardados al momento de cerrar la aplicación o web.

Es el adecuado ya que sin el ni siquiera se podría utilizar el react, sin este no se podría guardar datos a la marcha, es decir, se cierra la página o app y esta no guardará tus elecciones de filtros previas ni el buscador de texto.
